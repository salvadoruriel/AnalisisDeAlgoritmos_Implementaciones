Este trabajo presenta una colección de algoritmos con su descripción, pseudocódigo e implementación en JavaScript. Las fuentes originales de los algoritmos se mencionan en cada uno.

Las implementaciones en JavaScript son propias del autor (Salvador Aguirre) y se apegan lo más posible a las operaciones en pseudocódigo para no cambiar tiempos de ejecución. Estos tiempos son medidos exclusivamente en el algoritmo donde aplica, usando promesas en JavaScript para evitar congelar el navegador en operaciones tardadas, aunque según los valores ingresados por el usuario aun así podría atorarse la pestaña, a lo que se recomienda cerrar la página donde corre el algoritmo para interrumpir su ejecución.

Los algoritmos están separados según su temática general para facilitar la navegación en la página web y a través del Índice. Se puede observar el código usado en cada seccion abriendo la consola de desarrolladores (tecla F12 por lo general) y luego en:

  • Chrome o navegador Chromium: la pestaña "Sources" o "Fuentes".
  • Firefox: "Debugger" o "Depurador".

Ahí aparecen los archivos con el código fuente, y pueden seleccionarse los archivos JavaScript (terminados en ".js") para observar el código usado en cada parte.

Índice

  1. Algoritmos de ordenacion
  2. Algoritmos recursivos e iterativos
